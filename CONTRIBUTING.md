## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/yourusername/runanywhere-ai.git`
3. Create a branch: `git checkout -b feature/your-feature-name`
4. Make your changes
5. Test thoroughly
6. Commit: `git commit -m "Add: your feature description"`
7. Push: `git push origin feature/your-feature-name`
8. Open a Pull Request

## Development Guidelines

### Code Style
- Use TypeScript for type safety
- Follow existing code formatting
- Use meaningful variable names
- Add comments for complex logic

### Commit Messages
- Use clear, descriptive messages
- Start with: Add, Fix, Update, Remove, Refactor
- Example: "Add: CSV export functionality"

### Testing
- Test in Chrome and Edge (WebGPU)
- Test offline functionality
- Verify privacy mode works correctly
- Check mobile responsiveness
